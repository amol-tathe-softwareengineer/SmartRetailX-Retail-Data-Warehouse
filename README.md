# SmartRetailX - Retail Data Warehouse

## 📌 Project Overview
SmartRetailX is a Retail Data Warehouse project designed to manage and analyze retail sales data using ETL processes and SQL-based data modeling.

## 🏗️ Architecture
- Data Source: CSV Files (Customers, Products, Sales)
- ETL Layer: Python (Extract, Transform, Load)
- Database Layer: SQL Data Warehouse Schema
- Analytics: SQL Queries for reporting

## ⚙️ Technologies Used
- Python
- SQL
- Data Warehousing Concepts
- ETL Process

## 📂 Project Structure
- data/ → Raw datasets
- etl/ → ETL scripts (extract, transform, load)
- sql/ → Database schema & queries
- powerbi/ → Dashboard reports (if available)

## 🚀 How to Run
1. Load data from CSV files
2. Run ETL scripts in sequence
3. Execute SQL scripts for schema creation
4. Run analytical queries
